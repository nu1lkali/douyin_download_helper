# 抖音短视频去水印

**请求方式：** GET

**接口地址：** https://api.hk0.cc/api/douyin

**接口描述：** 抖音去水印，支持短视频、图集

## 请求参数

| 参数名 | 传递参数 | 传入位置 | 类型 | 参数说明 |
|--------|----------|----------|------|----------|
| url | string | query | 必传 | 视频链接 |
