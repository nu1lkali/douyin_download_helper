# 抖音无水印视频下载工具

基于Flutter开发的抖音无水印视频、图片下载应用。

## 功能特性

- 支持多种抖音链接格式解析
- 无水印视频下载
- 视频封面下载
- 一键下载（视频+封面）
- 下载历史记录
- 视频详情展示

## 安装依赖

```bash
flutter pub get
```

## 运行应用

```bash
flutter run
```

## 构建APK

```bash
flutter build apk --release
```

## 支持的链接格式

- 分享口令（包含短链接）
- 短网址：https://v.douyin.com/xxx
- 正常网址：https://www.douyin.com/video/xxx
- 发现页：https://www.douyin.com/discover?modal_id=xxx
